== Installation ==

Before installing, ensure that WooCommerce is installed and activated on your WordPress site.
Upload the plugin files to the /wp-content/plugins/plugin-name directory, or install the plugin through the WordPress plugins screen directly.
Activate the plugin through the 'Plugins' screen in WordPress.
Check if the wp_customer_feedback table is created in your database.
Go to Settings -> Permalinks and just save changes to ensure proper functionality.
